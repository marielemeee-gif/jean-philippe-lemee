# Site Jean-Philippe Lemée

Prototype éditorial autonome consacré à Jean-Philippe Lemée.

## Lancer le site

```bash
npm install
npm run dev
```

Puis ouvrir <http://localhost:3000>.

## Vérifications

```bash
npm run lint
npm run typecheck
npm run build
```

Les œuvres et photographies originales ne sont pas intégrées à cette première version, dans l’attente de la validation des droits et crédits.
